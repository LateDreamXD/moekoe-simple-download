# MoeKoe Simple Download
[MoeKoe Music](https://github.com/MoeKoeMusic/MoeKoeMusic) 插件, 简单地下载音乐到本地

## 食用方法
1. [下载](https://github.com/LateDreamXD/moekoe-simple-download/releases/latest)插件
2. 解压到 MoeKoe Music 插件目录
3. 重启 MoeKoe Music
4. 然后你就会发现音乐控制栏多了一个下载按钮
