[MoeKoe Music](https://github.com/MoeKoeMusic/MoeKoeMusic) 插件, 简单地下载音乐到本地

## 食用方法
1. [下载](https://github.com/LateDreamXD/moekoe-simple-download/releases/latest)插件
2. 解压到 MoeKoe Music 插件目录
  `%AppData%\moekoemusic\extensions`
3. 重启 MoeKoe Music

## 手动构建
```bash
# 克隆项目
git clone --depth 1 https://github.com/LateDreamXD/moekoe-simple-download.git
cd moekoe-simple-download
# 安装依赖
pnpm install
# 构建插件
pnpm build
```
<small>构建完成后, 插件文件位于 `dist` 目录</small>
